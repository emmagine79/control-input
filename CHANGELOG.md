# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-17

### Added

- One-click audio input switching from the menu bar.
- Preferred device with auto-switch -- automatically reverts to your chosen device when another device tries to take over.
- Device-type icons for built-in, Bluetooth, USB, and virtual audio devices.
- Theme support (System, Light, Dark) applied to the entire app.
- Launch at Login via SMAppService.
- Resizable settings window.
- CoreAudio device monitoring with real-time device connect/disconnect detection.
- Menu-bar-only operation (LSUIElement) -- no dock icon clutter.
- Native SwiftUI app targeting macOS 14+ (Sonoma).
